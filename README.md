## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* flask
  ```sh
  pip install flask
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/Pokeylooted/Shard.git
   ```
2. Install requirements using setuptools
   ```sh
   pip install .
   ```
